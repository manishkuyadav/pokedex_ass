PokéDex
Welcome to the PokéDex!

Open this menu anytime by clicking PokéDex

INSTALL APP

App can be used offline right away but for best experience let all the thumbnails (about 3MB only) load in background first.
Main images are loaded only when opened and once loaded they will be available for offline usage.

SETTINGS
Mega Evolution Animation

If enabled, the animation takes place once for each species after the base form's image have been downloaded.

ABOUT

v1.0
The current PokéDex is up to date with the data from the PokéAPI (Uptu Gen VII). Earlier this app used to call the API but now all the data is completely stored locally to facilitate offline usage. So this app works independently from the API.
Made with for Pokémon.
